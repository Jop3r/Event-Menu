# Event Menu (Garry's Mod Addon)

**Event Menu** — это мощный и удобный инструмент (аддон) для серверов Garry's Mod, предназначенный для управления внутриигровыми событиями (ивентами). Он предоставляет современный пользовательский интерфейс, систему приглашений и отображение Steam-аватаров игроков.

---

## 🚀 Основные возможности

- **Современный UI**: Красивое, интуитивно понятное меню администратора и игроков.
- **Интеграция со Steam API**: Автоматическая подгрузка и отображение реальных аватарок игроков через Steam (если аватар недоступен, генерируется заглушка с первой буквой ника).
- **Система приглашений**: Возможность выборочно приглашать игроков на ивент и отслеживать их статус.
- **Управление доступом**: Только указанные в конфигурации администраторы могут открывать панель управления и запускать события.
- **Оптимизированный сетевой код**: Разделение логики на `client`, `server` и `shared` для минимальной нагрузки на сервер.

---

## 🛠 Установка

1. Скачайте репозиторий в виде ZIP-архива или склонируйте его через Git.
2. Переместите папку `event_menu` в директорию аддонов вашего сервера:  
   `garrysmod/addons/event_menu`
3. Убедитесь, что структура папок выглядит так:  
   `garrysmod/addons/event_menu/lua/autorun/...`
4. Перезапустите сервер или выполните быструю смену карты.

## ⚙️ Настройка (Конфигурация)

Вся основная настройка производится в файле конфигурации:
`lua/autorun/shared/sh_event_menu_00_config.lua`

### 1. Настройка Steam API
Для корректной загрузки аватарок игроков требуется ключ Steam Web API.
1. Перейдите на страницу [Steam API Key](https://steamcommunity.com/dev/apikey) (потребуется авторизация).
2. Зарегистрируйте домен (можно любой) и получите свой ключ.
3. Откройте файл `sh_event_menu_00_config.lua` и вставьте ваш ключ:
   ```lua
   EventMenuConfig.SteamAPIKey = "ВАШ_КЛЮЧ_ЗДЕСЬ"
   ```
   *Примечание: Никогда не передавайте этот ключ посторонним.*

### 2. Добавление администраторов
Чтобы получить доступ к панели управления, вам нужно добавить свой SteamID в список разрешенных:
```lua
EventMenuConfig.Admins = {
    "STEAM_0:1:12345678", -- Пример
    "STEAM_0:0:87654321", -- Добавьте свой SteamID сюда
}
```

## 🎮 Использование в игре

Администраторы (чей SteamID внесен в список) могут использовать следующие чат-команды:
- `!eventmenu` — Открывает главную панель управления ивентами (настройки, приглашения).
- `!eventstart` — Позволяет быстро запустить ивент или открыть меню быстрого запуска.

Обычные игроки будут получать красивые UI-уведомления с приглашениями на события, которые они могут принять или отклонить.

## 📝 Разработчикам

Аддон структурирован по стандартным правилам GMod:
- `lua/autorun/client/` — UI интерфейс (панели, стили, шрифты) и клиентские хуки.
- `lua/autorun/server/` — Серверная логика, обработка сетевых сообщений (net messages) и запросы к Steam API.
- `lua/autorun/shared/` — Общие конфигурации и функции.
